# sudoku 2.0
画面で操作部分を追加  
懸賞を送るための合計値を計算

# 実行手順
## 必要なライブラリインストール
```
pip install -r requirements.txt
```
## 実行
```
python main.py
```
# 使い方
1. 決まっている数字を入力する
1. [計算]ボタンをクリックする
1. 結果が返ってきたら、懸賞用のセルをクリックする
1. 合計値を記入し懸賞に応募する
1. (^o^)
